# FFVII Compilation Lorebook

An interactive, single-page lorebook for the **Final Fantasy VII Compilation** —
covering the original game, Crisis Core, Before Crisis, Advent Children,
Dirge of Cerberus, and the Remake saga (Remake / Rebirth / the upcoming finale).

Open `index.html` in any browser — it's fully self-contained (no build step, no
dependencies, no external assets).

## Features

- **Timeline** — a chronological story spine from the Cetra (~2000 years ago) to
  the Geostigma aftermath, with events color-coded by source entry. Character
  names link straight into the relationship web.
- **Relationships** — an interactive force-directed graph of ~38 characters.
  Click a node to dim the rest and open a **dossier** with the character's
  temperament, role in the story, and clickable connections. Drag nodes to
  rearrange. Edge types: ♥ love · ⛓ family · ➜ mentor · ◆ ally · ⚔ enmity.
- **Compendium** — concept cards for the world's core forces: the Lifestream,
  Mako, Materia, Jenova, the Cetra, SOLDIER, Shinra, the WEAPONs, and more.
- **Spoiler toggle** — a "Classified / Declassified" switch (default off) seals
  the major twists behind Shinra-file redaction. Declassifying also reveals
  hidden relationship ties, so the web visibly grows as you unlock it.

## Design

A "Shinra Archive ⟷ Lifestream" aesthetic: a cold steel terminal frame around
luminous, living content. Mako-green glow, Shinra amber, and Jenova violet on a
steel-black ground; serif display + monospace utility + system body type.
